# 🌐 Staybnb Clone
Staybnb is a simple yet functional vacation rental web app built using HTML, CSS, JavaScript, and PHP. It allows users to browse property listings, make bookings, and enables hosts to list their properties for short stays — inspired by the Airbnb concept.

## 🚀 Features

- 🌐 login authentication for users and admin
- 📍 Displays the house details, price, location,etc...
- 🎨 Clean and responsive UI for desktop and mobile

🛠️ Technologies Used: 
- HTML5 / CSS3	Structure & styling
- JavaScript	Frontend interactivity
- PHP	Server-side scripting
- MySQL	Database for listings/bookings
- XAMP is used for Local development server
